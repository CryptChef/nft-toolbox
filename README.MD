## CRYPT-CHEF NFT - All in one NFT Toolbox

### Features

- Support to have multiple projects
- Init project from layers
    - Create **config.json** 
      - with,
        - Layer order configuration
        - Format asset size
        - Base URI
        - Forcecombination
        - Incompatible Traits
    - Default Layer order configuration included
- Generate project
    - Base on configuration file
    - Support for multiple workers
    - Check all required details at the start
      - Layers
      - Traits
      - Possible Unique asset count
- Get Rarity Table
- Get Summary
    - Number of layers
    - Number of Traits
    - Number of Uniques assets
    - Total collection size
